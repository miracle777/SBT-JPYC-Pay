**主要セクション**:

- ネットワーク自動設定実装パターン
- JPYC送金コード例
- SBT受取処理
- セキュリティ考慮事項
- テストネット設定
- 実装チェックリスト

**⭐ 最重要情報**:
```json
QRコード内に含まれるネットワーク情報:
- chainId: 137 (Polygon Mainnet) または 80002 (Amoy Testnet)
- jpycContractAddress: JPYC トークンアドレス
- recipientAddress: ショップウォレット
- amount: 支払い金額

これにより、スマホアプリ側で自動でネットワーク切り替え可能
```

## 💡 ドキュメント活用のコツ

### ✅ 効果的な使い方

1. **最初は全体像から**: README.md で全体を把握
2. **深掘りは必要に応じて**: 詳細ドキュメント参照
3. **実装時は手元に配置**: 実装ファイルと並べて確認
4. **定期的に更新確認**: 最新仕様をドキュメントから確認

### ⚠️ よくある間違い

❌ 最初から詳細仕様を読む → 全体像を失いやすい  
❌ ドキュメントを読まずにコード確認 → 前提知識を逃す  
❌ 古い情報を参照し続ける → 実装内容とズレが生じる  

✅ README → 概要ドキュメント → 詳細仕様 の順で読む  
✅ コードと並列参照 で理解を深める  
✅ 定期的に更新 を確認する
